About Product Backlog Developer

Languages Used: HTML5, CSS3, and JavaScript


Technologies Used: Backend: Google Firebase, Front-end: React JS


Description:

Product Backlog Developer was built as a full stack application. The backend, being Google Firebase, allows for realtime access to data. This allows multiple users to be connected to the same backlog, and changes made are reflected instantly. Google firebase also allows for users to be authenticated through the Firebase AuthUI. Each user is assigned a product backlog. Users can add stories or defects to this, and complete stories or defects as needed. The front-end using React, means that only changes need to be rendered. This results in quicker load times for the user.
